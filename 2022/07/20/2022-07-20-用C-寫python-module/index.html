用C++寫python module

2022-07-20

做這個小實驗的當天看到這個梗圖，相當呼應這個主題:

「其它的程式語言對你作力學是沒用的，作計算就三本柱 Python、C++、Fortran 足矣，連 C 都沒什麼用處。」Python在計算的速度上相較於C++非常的緩慢，這個demo很清楚地呈現了這件事。

1. 寫 .cpp/.h file。

2. 寫 CMakeList.txt。

3. Build the module。
