在C++中使用Pytorch模型

2022-08-26

計算流體力學(CFD)龐大的計算量加上對於即時流場預測的需求，計算效率是被重視的，於是python鮮少被使用，計算用的程式藉由C, C++ 以及 Fortran撰寫。我們實驗室其中一分支是研究計算效率的提升，方法上包括藉由平行計算，GPU計算，而藉由machine learning的方法來提昇計算效率也是研究對象之一；而常見的deep learning框架包括tensorflow以及pytorch都是藉由python撰寫，於是python輸出的model應用到C++撰寫的計算流力程式就需要耦合兩者的工具。Pytorch 官方所提供的"jit"可以將pytorch訓練出的model用到C++上，主要會分為三個步驟：1. pytorch model以"jit"格式輸出 2. C++中使用torch 以及 3. C++使用torch::jit

Python vs C++ 作為pytorch前端

藉由官方所提供的工具可以將整個過程，包括訓練過程都放到C++中，但若沒有強烈的需求官方建議不需要這麼做(link)，pytorch以python作為前端能夠維持叫良好的使用者體驗品質，而因為pytorch即便以python作為前端，需要計算需求時就會呼叫C++讓C++去完成，所以相比以C++作為前端，兩者在計算效率上並沒有太多差異。
